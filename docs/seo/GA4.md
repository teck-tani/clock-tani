## 2. 왼쪽 메뉴별 활용법

| 메뉴 | 용도 | 추천 확인 포인트 |
| --- | --- | --- |
| 보고서 (차트 아이콘) | 트래픽, 사용자 분석 | 가장 많이 볼 메뉴 |
| 탐색 (돋보기 아이콘) | 커스텀 분석 | 중급 이상일 때 활용 |
| 광고 (스피커 아이콘) | 광고 성과 | 광고 집행 시 활용 |

## 3. 처음이라면 이 순서로 보세요

### Step 1: 보고서 > 획득 > 트래픽 획득

- 사용자가 어디서 오는지 확인 (검색, 직접 접속, SNS 등)
- 가장 기본이면서 중요한 데이터

### Step 2: 보고서 > 참여도 > 페이지 및 화면

- 어떤 페이지를 많이 보는지 확인
- 인기 페이지 vs 안 보는 페이지 파악

### Step 3: 보고서 > 사용자 속성 > 인구통계 세부정보

- 누가 쓰는지 — 국가, 도시, 언어 등
- 스크린샷 보면 현재 South Korea 1명 활성

### Step 4: 보고서 > 참여도 > 이벤트

- 사용자가 무엇을 하는지 — 클릭, 스크롤, 페이지뷰 등

## 4. 실무 인사이트 얻는 팁

- **기간 비교**: 우측 상단 날짜에서 "비교" 체크 → 지난주 대비 이번주 변화 확인
- **이탈률 높은 페이지**: 참여도 > 페이지에서 이탈률 높은 페이지 → UX 개선 대상
- **실시간**: 배포 직후 실시간 탭으로 정상 접속 확인 용도로 활용
- **맞춤 이벤트**: 특정 버튼 클릭 등을 추적하려면 `gtag('event', ...)` 코드 추가 필요
